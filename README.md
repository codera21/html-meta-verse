# html-meta-verse
Fetching meta data of Html from url
## Usage
```
npm install 
npm run build
npm run meta -- --url=https://www.sastodeal.com/fumo-classic-faux-leather-bean-bag-xxxl-black-with-beans-fumo-sd-gg-002.html 
npm run serve 

```
