# html-meta-verse
Fetching meta data of Html from url
## Usage
```
npx tsc

npm run dev -- --url=https://www.sastodeal.com/fumo-classic-faux-leather-bean-bag-xxxl-black-with-beans-fumo-sd-gg-002.html
```
